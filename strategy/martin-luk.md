## Signature ideas

### Three-scanner selection
Build the daily candidate list from the **strongest prior-day performers**, then narrow to roughly
the **top ~2% of names**. Use this to read **sector rotation / thematic momentum** — which group is
leading right now (e.g. AI, quantum, crypto-adjacent, nuclear). Trade the leaders of the hot theme,
not random strength.

### Multi-timeframe stop tightening
Shift between **daily → hourly → 5-minute** charts to find the **tightest valid stop**. A smaller
stop lets you carry **bigger size at the same dollar risk** (still 1% for me). The entry timeframe
is chosen to minimize stop width, not to chase a prettier chart.

### Asymmetric R:R, low win rate
He runs a **low win rate (~20–30%) with ~5:1 reward:risk**. The edge is **asymmetry + discipline**:
cut losers instantly at the stop, let a few big winners pay for everything. Don't judge a setup by
"will it work" — judge it by R and by cutting fast when wrong.

### Strict cut-loss discipline
No averaging down, no widening stops, no hoping. If it hits the stop, it's out. Most trades are
small losers or scratches; the system only works if losers stay tiny.

## How I blend Luk + Qullamaggie
- **Selection:** Luk's prior-day-strength + theme read to build the list; Qullamaggie's setup
  definitions (breakout / EP) to time the entry.
- **Stops:** Luk's multi-timeframe tightening, always ≤ 1× ADR (Qullamaggie's cap).
- **Targets/exits:** Qullamaggie's trim-into-strength + trail the 10/20-day MA; size for the big-R
  asymmetry Luk leans on.
- **Mindset:** expect to be wrong often; protect capital; let winners run on the MA.
